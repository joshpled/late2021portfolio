### See locally

1. Clone the repo
   ```sh
   git clone https://github.com/joshpled/late2021portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally with NPM Start
   ```sh
   npm start
   ```
